# DETERMINE PROJECT

<h1> About Project </h1>
This project utilizes data from OCHIN Inc.
The goal of the project is to develop machine learning model to predict 5-year onset of developing type-2 diabetes meticullous given 1-year historic EHR data which includes:
1. Demographics
2. Medications
3. Lab results
4. Diagnoses code
5. Community vital signs (Zip code level SDoH features)
